<caniuse-embed> Element 使用示例

基础用法

嵌入脚本

<script src="https://unpkg.com/@cell-x/caniuse-embed-element/dist/caniuse-embed-element.iife.js"></script>

插入元素

特性名称获取详见：https://caniuse.lruihao.cn/

<caniuse-embed
  feature="css-grid"
  past="2"
  future="1"
  origin="https://caniuse.lruihao.cn"
  theme="auto"
>
</caniuse-embed>

特性名称（feature）

不指定 feature时：

过去版本（past）

显示过去 n 个版本，范围为 0 - 5，默认为 2

未来版本（future）

显示未来 n 个版本，范围为 0 - 3，默认为 1

[可选] 自定义数据源（origin）

例如：https://caniuse.pengzhanbo.cn

暗黑主题（theme）

可选：auto / light / dark
